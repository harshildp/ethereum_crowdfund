# Ethereum based Crowdfunding Site
A kickstarter esque crowdfunding website which utilizes ethereum and blockchain to correct the pervasive scamming problem with kickstarter

## Built with:
* Solidity
* Remix
* JavaScript
* Mocha
* Next.js
    * React
* Semantic UI
* Node
* Ganache
* Web3
* MetaMask
* Rinkeby Ethereum Test Network
